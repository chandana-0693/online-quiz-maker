# Online Quiz Maker

A premium, interactive, and fully responsive Online Quiz Maker Single-Page Application (SPA) built using vanilla HTML5, CSS3, and JavaScript.

## Features

- **Topic Categories & Filtering**: Select a topic to display and load relevant quizzes with smooth animations.
- **Dynamic Quiz Creator**: Build your own quizzes with questions, multiple-choice options, correct answer selections, and detailed explanations.
- **Interactive Quiz Player**: High-fidelity runner with progress indicator, visual selection states, and direct submissions.
- **Instant Detailed Feedback**: Results screen with animated circular progress indicator, XP gains, custom badges, and an interactive review sheet displaying incorrect answers and explanations.
- **User Authentication**: Mock system allowing registration, login, logout, and personalized dashboard tracking (XP level, attempts history, and user-created quizzes).
- **Premium UI Design**: Built with a sleek dark-glass theme (glassmorphism), customized gradients, beautiful typography, and smooth CSS transitions.
- **Completely Serverless**: Data persists locally in `localStorage`, making the app fully functional and portable without any local backend server configuration.

## Folder Structure

- `index.html`: Holds the single-page application structure.
- `styles.css`: Implements the visual styling rules, dark glass theme, and responsiveness.
- `app.js`: Houses the frontend routing, data management, mock database, and interactive rendering logic.

## How to Run

1. Open `index.html` directly in any web browser.
2. Alternatively, serve the project using a local dev server (e.g., VS Code Live Server or python `python -m http.server 8000`).
